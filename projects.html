Automatic Portfolio Generator

This project automates the creation of student portfolios. As data is entered into the database, a unique URL is generated using the student's roll number, which dynamically displays their profile page.

Features:
- Dynamic page generation using roll numbers.
- Centralized database for managing student data.
- Automatically updates portfolio when data changes.
- Responsive design for seamless viewing across devices.

Technologies:
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Database: MongoDB/MySQL
- Hosting: Vercel

By accessing the URL (example.com/rollno), the system fetches and displays the corresponding student's portfolio.

Enhancing Layering Effect with Interaction

To enhance the layering effect, add interactions that respond to scrolling. This will create depth and dynamic movement.


Apply a 'While Scrolling in View' interaction to the 'c-card' elements, ensuring it affects all cards with the same class. Set the smoothing value to 90% for smooth transitions.

Add a 'Scale' interaction starting at keyframe 40% (scale: 1) and ending at keyframe 100% (scale: 0.8). This makes the cards appear to recede as you scroll. Adjust the timing for a natural look.

For added impact, consider easing effects like 'ease-in-out' to create smooth transitions. These enhancements will make the stacking effect visually engaging and interactive.
